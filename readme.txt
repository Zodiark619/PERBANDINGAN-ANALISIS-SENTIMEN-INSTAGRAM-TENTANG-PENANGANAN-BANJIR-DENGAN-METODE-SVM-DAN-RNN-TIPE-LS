Instagram-Comments-Scraper-master merupakan program yang digunakan dalam proses scraping komentar 
instagram.

Program_preprocessing dijalankan menggunakan RStudio. Terdapat 3 file dataset komentar raw berupa 
comments,comments_banjir1, dan comments_banjir2. File databersih_hasilakhir merupakan dataset komentar
yang telah dibersihkan dan siap digunakan untuk program utama.

Program_utama, programutama_quisioner, dan programutama_svm dijalankan menggunakan anaconda dengan virtual 
environment. Tensorflow wajib diinstal di dalam virtual environment beserta seluruh library yang perlu diimport.
